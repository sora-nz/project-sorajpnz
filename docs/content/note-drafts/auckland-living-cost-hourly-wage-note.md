## メモ

NZ生活を考えるとき、最初はどうしても家賃に目が行きます。

でも、実際には家賃だけ見ても全然足りませんでした。

時給がいくらか、週に何時間働けるか、車を持つか、食費や通信費がどれくらいか、毎月いくら貯めたいか。こういう数字を一緒に見ないと、「生活できるか」ではなく「ギリギリ回っているだけなのか」が見えにくいです。

特にAucklandでは、車があると行動範囲は広がる一方で、燃料、保険、整備、rego/WOF、駐車場代が毎月の前提に入ってきます。

だからSoraJPNZでは、生活費をただの感想ではなく、前提を動かせる形で残していきたいです。

まずは下書きとして、時給・勤務時間・家賃・車・貯金目標からNZ生活の余白を見る記事を書いています。
